# CareerCompass AI

CareerCompass AI is a Flask-based Resume Analysis Platform that helps students identify career opportunities based on their resumes.

## Features

- Resume PDF Upload
- Skill Detection
- Career Recommendations
- Skill Gap Analysis
- Career Readiness Scores
- Learning Roadmaps
- PDF Report Generation
- User Authentication (Login/Signup)
- SQLite Database Integration
- User Report History
- Visual Analytics Dashboard

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Matplotlib
- ReportLab
- PDFPlumber
